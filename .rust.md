# Rust 学习仓库

本仓库用于系统性学习 Rust 编程语言，每个子项目对应一个技术章节，按编号顺序排列。

## 技术栈

- **语言**: Rust 2024 edition
- **构建工具**: Cargo
- **测试**: cargo-nextest、rstest、proptest
- **代码质量**: clippy、rustfmt、typos
- **依赖审计**: cargo-deny、cargo-audit
- **变更日志**: git-cliff

## 目录结构

```
Rust/
├── CLAUDE.md              # 全局约定（代码规范、错误处理、安全等）
├── .rust.md               # 本文件 — 项目架构说明
├── README.md              # 仓库介绍与环境配置指南
├── .gitignore             # Git 忽略规则
├── 000-template/          # 项目模板 — 新项目的起点
│   ├── CLAUDE.md          # 项目约定
│   ├── README.md          # 项目说明
│   ├── Cargo.toml         # 依赖配置
│   └── src/
│       └── main.rs        # 入口文件
├── 001-xxx/               # （待添加）
└── ...
```

## 开发环境

### 前置条件

```bash
# 安装 Rust
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh

# 安装工具链
cargo install cargo-generate      # 项目模板生成
cargo install --locked cargo-deny # 依赖审计
cargo install typos-cli           # 拼写检查
cargo install git-cliff           # 变更日志
cargo install cargo-nextest --locked # 增强测试
pipx install pre-commit           # 提交前检查
```

### 常用命令

```bash
# 进入某个项目
cd 000-template

# 构建
cargo build

# 运行
cargo run

# 测试
cargo nextest run

# 格式化
cargo +nightly fmt

# 代码检查
cargo clippy -- -D warnings

# 依赖审计
cargo deny check
```

## 新建项目流程

1. 复制 `000-template` 目录，重命名为 `NNN-项目名`
2. 修改 `Cargo.toml` 中的 `name` 字段
3. 更新项目内的 `README.md` 和 `CLAUDE.md`
4. 在根目录 `README.md` 的项目列表中添加新条目
5. 开始编写代码
